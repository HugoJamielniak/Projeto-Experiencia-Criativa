Sobre a matéria

Neste curso de Fundamentos de Sistemas Ciberfísicos, foram desenvolvidos diversos projetos que integraram componentes físicos e digitais para resolver problemas reais. As atividades envolveram o uso de sensores, atuadores e programação para criar sistemas inteligentes capazes de monitorar e controlar ambientes, enfatizando a interação entre o mundo físico e o computacional.

Projetos

Projeto sobre Identificação de crimes e alerta a polícia/segurança privada

Este projeto propõe um sistema de alarme inteligente com ESP32 para detectar invasões em locais vulneráveis, como ONGs e casas de repouso. Utilizando sensores de presença e envio de alertas via MQTT, a solução garante segurança eficiente, acessível e de fácil implementação.

Projeto sobre Operações de Bitwise e shift de bits

Este projeto aborda as operações bitwise e shift de bits, explicando seus conceitos fundamentais e exemplificando pelo menos cinco operações de cada tipo.

Projeto sobre Chipsets e funcionalidades.

Este projeto envolve uma pesquisa detalhada sobre chipsets atuais, abordando sua definição, principais características e diferenças em relação à BIOS. Também inclui a análise de exemplos modernos e sua importância para o desempenho dos sistemas computacionais.
